# Ailin Zhang Website Clone

This project recreates the previous Weebly personal website as a GitHub Pages site.

## Files

- `index.html`: About page
- `research.html`: Research page
- `teaching.html`: Teaching page
- `styles.css`: local overrides for the cloned theme
- `site.js`: mobile navigation toggle
- `assets/`: downloaded CSS, fonts, images, and theme assets

## Notes

- The site is now self-contained inside this folder and does not need to load theme assets from Weebly.
- Weebly-only ads, analytics, and account scripts were removed.

## Publish with GitHub Pages

1. Create or open a GitHub repository.
2. Put these files in the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Choose `Deploy from a branch`.
5. Select the `main` branch and `/root`.
6. Save and wait for the site URL to appear.
